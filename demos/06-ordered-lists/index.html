The ordered list (ol) element is always paired with list item (li) elements as children. In ordered lists, list items have numbers in front of them by default.
  1. List Item 1
  2. List Item 2
  3. List Item 3
  4. List Item 4
  5. List Item 5
Ordered lists have default margin and padding applied to them. They are frequently used to display information that has an order to it, such as rankings or steps in a recipe.
